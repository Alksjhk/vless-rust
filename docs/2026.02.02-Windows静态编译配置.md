# 2026.02.02-Windows静态编译配置

## 修改内容

### 静态链接配置
- 添加 `.cargo/config.toml` 配置文件
- 设置 `rustflags = ["-C", "target-feature=+crt-static"]`
- 实现 Windows 版本 CRT 静态链接

### GitHub Actions 工作流
- 修改 Windows 构建配置：`link_type` 从 `dynamic` 改为 `static`
- 添加 Windows 静态链接配置步骤（PowerShell 语法）

## 效果

- Windows 版本编译为单一 exe 文件，零依赖运行
- 不需要安装 Visual C++ Redistributable
- 本地编译和 CI 编译结果一致
- 与 Linux 版本保持相同的静态链接策略

## 技术细节

CRT 静态链接后，Windows 可执行文件只依赖 `KERNEL32.dll`，不再依赖 MSVC 运行时库（msvcr*.dll、vcruntime*.dll 等）。
