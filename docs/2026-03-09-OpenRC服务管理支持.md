# 2026-03-09 OpenRC 服务管理支持

## 更新内容

### 新增功能

- **OpenRC 服务管理**：支持 Alpine Linux、Gentoo 等 OpenRC 发行版
- **自动检测初始化系统**：程序自动识别 systemd 或 OpenRC
- **统一服务管理入口**：`--init` 和 `--remove` 参数自动适配系统

### 改进优化

- 重构服务管理模块，抽象 `InitSystem` 枚举
- OpenRC 服务安装需要 root 权限，systemd 无需 root
- 完善服务安装后的日志查看提示

## 使用方法

### 自动检测（推荐）

```bash
# 自动检测并安装对应服务
./vless --init

# 自动卸载
./vless --remove
```

### OpenRC 服务管理

```bash
# 安装服务（需要 sudo）
sudo ./vless --init

# 卸载服务（需要 sudo）
sudo ./vless --remove

# 查看状态
rc-service vless-rust-serve status

# 查看日志
tail -f /var/log/vless-rust-serve.log
```

## 技术细节

**初始化系统检测优先级**：
1. 检测 `/run/systemd/system` → Systemd
2. 检测 `/run/openrc` 或 `/sbin/openrc` → OpenRC
3. 无匹配 → 返回错误

**OpenRC 服务特性**：
- 服务文件位置：`/etc/init.d/vless-rust-serve`
- 日志文件：`/var/log/vless-rust-serve.log` 和 `.err`
- 需要运行 `checkpath` 确保目录和日志文件权限
