# 产品需求文档 (PRD)

## 产品定位

VLESS-Rust 是一款高性能、轻量级的 VLESS 协议代理服务器，专为需要稳定、高速代理传输的场景设计。

## 目标用户

- 需要搭建私有代理服务的运维人员
- 追求高性能代理体验的技术用户
- 需要多用户管理的团队和组织
- 对代理服务稳定性有高要求的用户

## 核心功能需求

### 1. 协议支持
- **VLESS 协议**: 完整支持版本 0 和版本 1
- **传输协议**: TCP 和 WebSocket 双模式
- **UDP 支持**: UDP over TCP 传输
- **地址类型**: IPv4、IPv6、域名

### 2. 认证机制
- **UUID 认证**: 多用户 UUID 唯一标识
- **用户管理**: 支持配置多个用户
- **邮箱标识**: 可选用户邮箱字段

### 3. 代理功能
- **TCP 代理**: 高效双向数据转发
- **UDP 代理**: UDP over TCP 封装传输
- **WebSocket 代理**: 支持 WS 协议传输
- **目标连接**: 自动连接目标服务器

### 4. HTTP API
- **信息页面**: 服务器状态展示
- **链接生成**: 自动生成 VLESS 链接
- **多协议支持**: TCP 和 WS 链接生成
- **Base64 编码**: 链接编码输出

### 5. 配置管理
- **配置向导**: 交互式配置创建
- **JSON 配置**: 结构化配置文件
- **性能参数**: 可调整缓冲区和超时
- **配置验证**: 自动检查配置有效性

### 6. 服务管理
- **Systemd 支持**: Linux systemd 用户服务
- **OpenRC 支持**: Alpine/Gentoo 系统服务
- **自动检测**: 智能识别系统初始化系统
- **服务状态**: 服务安装、卸载、状态查询

### 7. 运维功能
- **TUI 界面**: 终端日志实时显示
- **优雅关闭**: 信号处理和平滑终止
- **公网 IP**: 自动获取服务器公网地址
- **日志管理**: 分级日志输出

## 非功能需求

### 性能要求
- **高并发**: 支持大量并发连接
- **低延迟**: TCP_NODELAY 优化
- **高吞吐**: 128KB 默认缓冲区
- **内存高效**: mimalloc 高性能分配器

### 可靠性要求
- **稳定性**: 长期运行无内存泄露
- **容错性**: 连接异常自动处理
- **原子写入**: 配置文件安全写入
- **优雅降级**: 错误情况下安全退出

### 安全性要求
- **认证安全**: UUID 唯一认证凭据
- **路径验证**: 防止路径遍历攻击
- **日志安全**: 不记录敏感信息
- **配置安全**: 文件权限管理

### 易用性要求
- **零依赖**: 静态链接，单文件部署
- **跨平台**: 支持 Windows/Linux/macOS
- **配置简单**: JSON 格式，向导引导
- **文档完善**: README 和技术文档

### 可维护性要求
- **模块化**: 清晰的模块划分
- **代码规范**: Rust 标准编码规范
- **错误处理**: 完善的错误提示
- **日志记录**: 详细的运行日志

## 部署要求

### 运行环境
- **操作系统**: Windows 7+、Linux (glibc/musl)、macOS 10.15+
- **架构支持**: x86_64、ARM64、ARMv7
- **依赖**: 无（静态链接）
- **权限**: 绑定端口需管理员/root 权限

### 资源要求
- **内存**: 最小 10MB
- **磁盘**: 约 1MB（可执行文件）
- **CPU**: 无特殊要求
- **网络**: 需要网络连接

### 部署方式
- **单文件部署**: 直接运行可执行文件
- **服务部署**: Linux 服务自动管理
- **容器部署**: 支持 Docker 等容器
- **配置文件**: JSON 格式配置文件

## 扩展规划

### 短期规划
- TLS 加密支持
- 更多传输协议（HTTP/2、gRPC）
- 流量统计功能
- Web 管理界面

### 长期规划
- 分布式部署支持
- 动态配置加载
- 插件系统
- 性能监控面板

## 成功指标

- 单连接带宽达到千兆网络水平
- 并发连接数支持 1000+
- 长期运行内存稳定无泄露
- 部署时间 < 5 分钟
- 配置错误率 < 1%
