# 2026-02-25-TUI终端界面

## 概述

本次更新添加了 TUI（终端用户界面），提供可视化的服务器状态监控和日志查看功能。

## 新增功能

### TUI 终端界面
- 新增 `src/tui.rs` 模块，实现日志收集和界面渲染
- 新增 `src/version.rs` 模块，管理版本信息和服务器状态
- 固定头部显示服务器状态（监听地址、协议、用户数、缓冲区大小）
- 可滚动日志区域（最多 1000 条）
- 自动滚动模式（新日志自动显示在底部）

### 交互控制
- `q` / `Esc`: 退出程序
- `↑` / `↓` 或 `k` / `j`: 上下滚动日志
- `Page Up` / `Page Down`: 快速翻页
- `Home`: 跳转到顶部
- `End`: 跳转到底部并重新启用自动滚动

### TUI 禁用选项
- 命令行参数: `--no-tui`
- 环境变量: `DISABLE_TUI=1`

## 依赖更新

### 新增依赖
- `ratatui`: TUI 界面库
- `crossterm`: 跨平台终端操作
- `chrono`: 时间处理

### build-dependencies
- `winres`: Windows 资源文件生成
- `serde`: 序列化（build 阶段）
- `serde_json`: JSON 解析（build 阶段）

## 使用方式

### 默认运行
```bash
cargo run
```

### 禁用 TUI
```bash
# 命令行参数
./vless.exe --no-tui

# 环境变量
DISABLE_TUI=1 ./vless.exe
```

## 技术细节

### 架构设计
- 主线程运行 TUI，后台线程运行服务器
- 使用 channel 在线程间传递日志
- 支持优雅关闭（检查停止标志）

### 性能优化
- 最多缓存 1000 条日志
- 用户手动滚动时禁用自动滚动
- 使用 `try_recv()` 非阻塞接收日志
